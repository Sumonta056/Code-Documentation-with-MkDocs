## Installation

To install MkDocs, run the following command:

- Create a virtual environment

```bash
python -m venv venv
```

- Activate the virtual environment

```bash
.\venv\Scripts\Activate
```

- Install MkDocs

```bash
pip install mkdocs-material
```

- Install Server

```bash
mkdocs new .
```

- Run the server

```bash
mkdocs serve
```
